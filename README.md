# URGENCIAS-UDC

Guía rápida de repaso sobre principios de ACLS, ATLS, TOXIDROMES, entre otras urgencias relevantes en la practica clínica.

## Objetivo

Servir como una herramienta de estudio y repaso para reforzar los conceptos fundamentales de RCP y ACLS.

## Página web

La guía puede consultarse como página web a través de GitHub Pages una vez activada la publicación del repositorio.

## Aviso importante

Este material tiene fines educativos y de repaso. No sustituye la formación oficial, la certificación ACLS ni los protocolos institucionales vigentes.
